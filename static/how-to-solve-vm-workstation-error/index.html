How to Solve the vmci driver error in VMware

Aug 30, 2016 0 Comments

Some of you migth get the following error while trying to run an existing VM workstation with VMworkstation under Windows 10:

The solution is simple, just open the configuration file of the VM in notepad. (look for a file named like TheNameOfYourVM.vmx in your VM folder).

Edit the following line:

vmci0.present = "TRUE"

And change it to False:

vmci0.present = "FALSE"

Problem should be resolved now.
