How to recover deleted files from USB in Linux

May 07, 2016 0 Comments

Some of you might know that by default, most Linux distros don't erase the files in a USB stick when we delete them. Actually, a hidden _trashes folder is created, with all the 'deleted' files inside.
But what happen when that folder is also removed? How to recover our precious files?

It happened that someone by mistake erased ALL the files of my USB stick. Lucky for me, I found testdisk, an open source software hero that recover every deleted file from a disk, or in my case, a USB stick.

To install testdisk in Ubuntu, just open the Terminal and type:

sudo apt-get install testdisk

There is also photorec, which is installed with testdisk. To use it, just type photorec in the terminal:

Select the disk you want to scan, and follow the steps. You may have to wait several minutes (in my case, almost two hours). But believe me, it's worth the time.

Photorec also works in Windows and Mac. You can check The official website to get more information.
